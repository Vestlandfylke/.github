# Velkommen til Vestland fylkeskommune på GitHub

![Vestland fylkeskommune](https://www.vestlandfylke.no/theme-vestland/images/logo/logo_vestland.svg)

## Om oss

**Vestland fylkeskommune** er Noregs tredje største fylkeskommune, med ansvar for vidaregåande opplæring, fylkesvegar, kollektivtrafikk, tannhelse, kultur og regional utvikling i Vestland fylke.

Vi utviklar digitale løysingar og deler open kjeldekode for å:
- Forbetre tenester innan opplæring og utdanning
- Effektivisere kollektivtrafikk og samferdsel
- Modernisere offentleg forvaltning
- Samarbeide med andre fylkeskommunar og kommunar

## Våre prosjekt

Her finn du kode, verktøy og ressursar som vi har utvikla. Vi trur på open kjeldekode og deling av løysingar for å skape betre offentlege tenester.

### Teknologiar

Vi nyttar moderne webteknologiar og skytenester, med hovudfokus på Microsoft Azure som plattform.
